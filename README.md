# 💰 Segundo Cérebro para Orçamento Pessoal

## Sobre o projeto

Este projeto foi desenvolvido utilizando o NotebookLM como um segundo cérebro para organizar informações sobre orçamento pessoal.

Seu objetivo é reunir conteúdos, resumos e referências em um único local, facilitando o estudo e a consulta sobre educação financeira.

## Objetivos

- Organizar conhecimentos sobre finanças pessoais;
- Auxiliar no planejamento financeiro;
- Centralizar materiais de estudo;
- Incentivar uma melhor gestão do orçamento.

## Ferramentas utilizadas

- NotebookLM
- GitHub
- Markdown
