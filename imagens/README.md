# 🖼️ Imagens

Esta pasta contém os recursos visuais utilizados no projeto **Segundo Cérebro para Orçamento Pessoal**.

## Conteúdo

- **Mapa Mental**: representação visual dos principais conceitos relacionados ao orçamento pessoal, planejamento financeiro e organização das informações.

## Objetivo

O mapa mental auxilia na compreensão da estrutura do projeto e na visualização das relações entre os temas abordados no miniguia.
